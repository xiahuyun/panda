# Panda Mobile 状态矩阵

## 1. 登录页

| 状态 | 触发条件 | UI 表现 | 用户操作 |
| --- | --- | --- | --- |
| `loading` | 提交登录请求后 | 按钮文案 `登录中...` 且禁用 | 等待 |
| `content` | 初始进入或请求结束 | 输入手机号、验证码 | 提交 |
| `error` | 登录失败 | 状态文本显示错误 | 修改输入后重试 |
| `empty` | 不适用 | 无 | 无 |

## 2. 缘分墙

| 状态 | 触发条件 | UI 表现 | 用户操作 |
| --- | --- | --- | --- |
| `loading` | 拉取推荐中 | 文案 `正在寻找合适的人...` | 等待 |
| `content` | 有推荐卡片 | 展示卡片与推荐理由 | 换一个 / 开始聊天 |
| `empty` | 无推荐数据 | 空态文案 + `重新获取` | 重试 |
| `error` | 请求失败 | 错误文案 + `重试` | 重试 |

## 3. 聊天页

| 状态 | 触发条件 | UI 表现 | 用户操作 |
| --- | --- | --- | --- |
| `loading` | 拉取历史消息中 | `正在加载聊天记录...` | 等待 |
| `content` | 消息加载成功 | 气泡列表 + 输入栏 | 发送消息 |
| `error` | 拉取失败 | 错误文案 + `重试` | 重试 |
| `empty` | 暂无消息 | 空态提示 + 输入栏 | 发首条消息 |

## 4. 发送消息子状态

| 状态 | 触发条件 | UI 表现 | 用户操作 |
| --- | --- | --- | --- |
| `sending` | 点击发送后 | 按钮显示 `发送中` 且禁用 | 等待 |
| `success` | API 成功 | 新消息追加到列表 | 继续发送 |
| `failure` | API 失败 | 草稿保留 + 错误提示 | 再次发送 |
