# Panda Mobile 交互流程图 (V0.2)

## 1. 主流程（MVP）

```mermaid
flowchart TD
  A["打开小熊交友 App"] --> B["进入登录页<br/>小熊交友 - 吃苹果熊猫 Logo 登录页"]
  B --> C["输入手机号 + 6位验证码"]
  C --> D["点击 登录/注册"]
  D --> E["本地 Mock 校验"]
  E --> F{"验证码 == 123456 ?"}
  F -- "否" --> G["弹窗：验证码错误"]
  G --> H["3秒后自动关闭"]
  H --> B
  F -- "是" --> I["登录成功"]
  I --> J["进入 缘分墙 - 活跃状态 (普通用户版)"]
```

## 2. 异常处理

1. 验证码错误：弹窗显示 `验证码错误`，3 秒自动关闭。
2. mock service 异常：沿用同一错误提示策略，停留登录页重试。

## 3. 说明

1. 当前流程仅覆盖登录与进入缘分墙活跃态。
2. MVP 不依赖真实后端，不持久化登录状态。
3. 聊天、匹配策略等均不在本轮范围内。
